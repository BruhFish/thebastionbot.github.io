What features do you think an ideal Discord Bot should have?

Administration, Moderation, Music, Game Server Stats, Player Stats, Searches, Games, User Profiles, User Levels, Virtual Currencies, Humor...
Think no more because Bastion BOT can do it all, and much more. It will even talk with you! Ain't that cool?

Get aboard.

If you want to get the chill of Bastion or if you want to play around with it, before you add it to your Discord Server, you can join the Bastion Bot: Discord Server and test it.

To get a complete list of what Bastion Bot can do, see the commands list.

We've got your back.

If you ever face any problems with Bastion or you don't understard something, or if you need any kind of help with it, just head over to the #help channel in our Discord Server, and ask for help. Everyone will be more than willing to help you.

star Pro Tip:

Do you like giveaways? We like it too. Everyone does. So we also hold some sort of giveaway events here too. Just keep an eye on the #announcements channel of our Discord Server.

Other than asking questions about Bastion, you can chat around with anyone, play, test Bastion, or whatever (not exactly whatever) you want, you can do it there.

We listen to you.

If you have an amazing suggestion that you think Bastion Bot should have, please post your suggestion in the #suggestions channel of our Discord Server or make a new issue in the GitHub Repository.
We'll be glad to implement that if that's something that is going to help everyone and is possible to do.

If you found a bug in Bastion Bot, please post your bug in the #bugs channel of our Discord Server or make a new issue in the GitHub Repository.
I can't say how thankful I am. I will fix that as soon as possible.

You get 25 Bastion Currencies for reporting a valid bug or if your suggestion gets implemented.

Come collaborate.

If you have a way around JavaScript & NodeJS and if you want to contribute to the code of Bastion, just Fork the GitHub Repository, make your changes or add something new, and create a Pull Request.
I will go through it and implement it if everything seems cool.

You will also get a cool Role in our Discord Server to show your pride!

Bring Bastion to your language.

If you want Bastion to be able to interact with you in your native language, you can contribute to the translation of Bastion Bot. And as soon as the translation of a language is done, and the translations are approved, it will be available in Bastion Bot.

If you want to translate Bastion to a language that is not already listed, @mention me (k3rn31p4nic#8383) in the #chat channel of our Discord Server or send me a direct message, I will add it to the list and you can then start translating.
