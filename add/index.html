Add Private Bastion Bot to your Discord Server

Private Bastion is the self hosted version of Bastion Bot, i.e. you host an instance of Bastion by yourself (for yourself) on a computer/server.

If you want to host Bastion by yourself, so that you can experience Bastion with complete access to each and every bit of it and can customize it to your liking, read the installation guide.
